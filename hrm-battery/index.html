HRM

A companion to the heart-rate-server example. Allows you to receive notifications for the Heart Rate characteristic and read the Battery Level characteristic and prints values in the console below.

To use:

  1. Run the heart-rate-server example on a device
  2. Click the "Connect" button below
  3. Select the device named "Moddable HRM" from the popup
  4. Receive notifications for the Heart Rate characteristic by clicking the "Start notifications" button below; read the Battery Level characteristic by clicking the "Read battery level" button
